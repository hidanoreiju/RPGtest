# 🎮 RESONANCE -追憶と再会の響き-

「記憶」と「音」が交錯する世界を舞台にした、本格2DアドベンチャーRPGのプロジェクトです。

---

## 📖 作品コンセプト
本作は、失われた「追憶」の断片を探し求め、人々の心と「再会」していく物語を描いています。
プレイヤーは広大なフィールドを探索し、様々なキャラクターとの交流を通じて、物語の核心へと迫ります。

---

## 🚀 実行方法 (How to Play)
特別なソフトのインストールは不要です。以下の手順でブラウザからすぐに試遊いただけます。

1. **ダウンロード:** 画面右上の「Code」ボタンをクリックし、「Download ZIP」を選択して解凍します。
2. **実行:** フォルダ内にある **`index.html`** をダブルクリックして、お使いのブラウザ（Chrome/Edge等）で開いてください。
3. **開始:** ブラウザ上にゲーム画面が表示されたら、操作を開始してください。

---

## 🕹 操作方法
本作では、プレイヤーがゲーム世界へスムーズに介入できるよう、極めてシンプルな操作系を採用しています。

| アクション | 操作方法 | 備考 |
| :--- | :--- | :--- |
| **移動** | 方向キー (↑↓←→) / WASD | キャラクターの移動に使用します。 |
| **会話・調査** | **対象の上に重なる（乗る）** | NPCや調べたいオブジェクトに**直接重なる**ことで、自動的にイベントや会話が開始されます。 |
| **メニュー/戻る** | Esc / Xキー | アイテムの確認や設定変更に使用します。 |

> **開発ノート:** > 「ボタンを押す」というアクションを排除し、接触によって心が通じ合う演出意図を込めた仕様です。

---

## ⚔️ プレイフロー
1. **探索 (Search):** フィールド内を移動し、気になる人物や場所を探します。
2. **共鳴 (Resonance):** 対象に接触（重なる）して情報を取得。物語を進める鍵を手に入れます。
3. **進行 (Progress):** 得られた情報をもとに次の目的地を目指し、物語を完結へと導きます。
